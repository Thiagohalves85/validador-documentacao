# Como usar

## Objetivo

A aplicação ajuda a revisar documentos fiscais antes do uso operacional, destacando erros estruturais, rejeições de negócio e avisos.

## Fluxo básico

1. Acesse a interface web.
2. Faça login com o usuário cadastrado.
3. Cole ou envie o conteúdo do documento.
4. Confirme o tipo identificado pela aplicação.
5. Execute a validação.
6. Revise rejeições, alertas e mensagens de sucesso.
7. Se houver recursos complementares para o documento, execute-os.
8. Valide novamente após qualquer ajuste.

## Formas de entrada

- colar o conteúdo do documento
- enviar arquivo
- arrastar arquivo para a área de edição

## Recursos complementares

Dependendo do documento, a aplicação pode oferecer:

- geração de campos complementares de RTC
- cálculo de base para NFS-e
- geração de documento auxiliar em PDF

## Boas práticas de uso

- corrigir primeiro a rejeição principal
- validar novamente após cada ajuste relevante
- usar exemplos fictícios em treinamentos e homologação
- revisar o conteúdo final antes de distribuição ou transmissão por outros sistemas
