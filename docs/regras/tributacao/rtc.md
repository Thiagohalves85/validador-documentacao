# RTC

## Objetivo

Registrar a visão funcional do processamento complementar ligado à Reforma Tributária do Consumo.

## Quando se aplica

Quando o documento possui dados suficientes para cálculo ou preenchimento complementar no fluxo atual.

## Exemplos de dados normalmente esperados

- classificação tributária compatível
- base de cálculo identificável
- informações do item ou serviço
- município e data do fato gerador quando aplicável
