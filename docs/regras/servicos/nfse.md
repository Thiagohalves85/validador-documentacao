# NFS-e Nacional

## Objetivo

Consolidar a visão funcional das regras de negócio ligadas ao fluxo de NFS-e Nacional.

## O que costuma ser analisado

- dados essenciais do serviço
- identificação das partes envolvidas
- coerência dos valores informados
- consistência mínima para cálculo complementar quando aplicável

## Recurso complementar

Para cálculo de base em NFS-e, a aplicação depende da presença dos dados essenciais do serviço e do contexto da operação.
