# Erros comuns

## Situações recorrentes

- campo obrigatório ausente
- identificação inconsistente entre as partes
- valor incompatível com total
- código fiscal incompatível com o contexto
- data fora da regra esperada
- XML incompatível com o documento informado

## Como agir

1. localizar a primeira inconsistência bloqueante
2. corrigir o conteúdo de origem
3. validar novamente
