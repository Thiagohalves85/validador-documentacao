# Validação

## Objetivo

Explicar o que acontece quando o usuário executa a validação.

## O que a aplicação confere

- estrutura esperada do documento
- consistência dos dados informados
- presença de campos obrigatórios
- compatibilidade entre códigos e contexto do documento

## Saídas possíveis

- aprovação no fluxo atual
- rejeição bloqueante
- alerta para revisão
