# Cadastro e login

## Objetivo

Explicar como o usuário acessa a aplicação.

## Formas de acesso

- cadastro local com nome, usuário e senha
- acesso com conta Google quando a integração estiver habilitada

## Comportamento da sessão

- a sessão permanece ativa até o vencimento configurado
- o usuário pode optar por manter o acesso neste dispositivo
- o navegador pode salvar as credenciais para uso futuro

## Quando usar

Este fluxo deve ser realizado antes de validar documentos, consultar histórico ou usar integrações protegidas.
