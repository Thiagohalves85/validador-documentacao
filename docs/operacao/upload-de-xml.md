# Upload de XML

## Objetivo

Descrever como o documento entra na aplicação para análise.

## Formas de envio

- colar o conteúdo manualmente
- selecionar arquivo local
- arrastar o arquivo para a área de edição

## Cuidados de uso

- conferir se o arquivo corresponde ao documento esperado
- evitar uso de arquivos com dados reais em treinamentos públicos
- revisar se o conteúdo foi carregado integralmente antes da validação
