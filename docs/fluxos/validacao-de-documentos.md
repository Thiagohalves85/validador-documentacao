# Validação de documentos

## Objetivo

Explicar o fluxo funcional de validação desde a entrada do documento até a leitura do resultado.

## Fluxo básico

1. Acesse a interface web.
2. Faça login com o usuário cadastrado.
3. Cole, envie ou arraste o conteúdo do documento.
4. Confirme o tipo identificado pela aplicação.
5. Execute a validação.
6. Revise rejeições, alertas e mensagens de sucesso.
7. Se houver recurso complementar para o documento, execute-o.
8. Valide novamente após qualquer ajuste.

## Formas de entrada

- colar o conteúdo do documento
- enviar arquivo
- arrastar arquivo para a área de edição

## Resultado esperado

Ao final do fluxo, o usuário entende se o documento:

- está aprovado no fluxo atual
- contém rejeições bloqueantes
- possui alertas que pedem revisão
