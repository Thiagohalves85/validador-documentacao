# Tratamento de rejeições

## Objetivo

Orientar a leitura funcional das rejeições retornadas pela aplicação.

## Sequência recomendada

1. localizar a primeira rejeição bloqueante
2. identificar o campo ou contexto citado
3. corrigir o documento de origem
4. validar novamente
5. revisar alertas remanescentes

## Boas práticas

- corrigir primeiro a rejeição principal
- evitar ajustes simultâneos sem nova validação
- usar exemplos fictícios em treinamento e homologação
- revisar o documento final antes de distribuição ou transmissão
